{%endblock%} {%block help_header%} Exporting a ZFS pool {%endblock%} {%block help_body%}

Use the ZFS pool export when you want to move a set of disks from one system to another. Exporting a pool will cause the kernel to flush all pending data to disk, writes data to the disk acknowledging that the export was done, and removes all knowledge that the storage pool existed in the system. Please ensure that this is actually intended before continuing.
